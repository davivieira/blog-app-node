## How to run

1. Install dependencies:
```
$ npm install
```

2. Set `NODE_ENV` in environment variables (unix):
```
$ export NODE_ENV=dev
```

3. Running:
```
$ npm start
```

or (recommended), you can use gulp to run tests + lint + start the app:
```
$ gulp watch
```
